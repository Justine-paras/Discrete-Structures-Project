### **1. Campus Navigation System**
Uses **Graph Theory + Dijkstraâ€™s Algorithm** to compute the shortest walking path between major DLSU-D buildings.

### **2. Scholarship Eligibility Checker**
A logic-based evaluator that determines student eligibility using **propositional logic**.

- Requirements mapped to Boolean variables (GPA, units, citizenship, varsity, etc.)  
- Logical formulas (conjunction + implication)  
- Clear feedback on missing requirements  
- Supports multiple scholarship types  

### **3. Smart Parking Allocation**
A lightweight parking manager using **Set Theory**.

- Parking lots represented as sets  
- Availability computed using set operations  
- Add/remove occupied slots  
- Simple interface for viewing and updating parking status
